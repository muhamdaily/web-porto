**Smart Ribbed Smoked Sheet Drying System - PTPTN XII Kebun Kalisanen** is an IoT-integrated web platform designed to manage and optimize the rubber smoking and drying process. The system provides real-time monitoring and automated control to ensure consistent quality, efficiency, and reliability in ribbed smoked sheet (RSS) production.

---

## The Story Behind Smart Ribbed Smoked Sheet Drying System

Traditional rubber smoking processes often rely on manual monitoring, which can lead to inconsistent temperatures, humidity levels, and drying durations. These inconsistencies may affect the final quality of the ribbed smoked sheet and reduce operational efficiency.

This project was developed to integrate IoT technology into the rubber drying workflow. By collecting real-time data from on-site sensors, the system allows operators to monitor conditions and automatically adjust temperature, humidity, and smoking duration based on predefined parameters. The result is a more controlled, data-driven process that improves product consistency while reducing human error.

---

## Key Features

* Real-time monitoring of temperature and humidity via IoT sensors
* Automated control of smoking and drying parameters
* Centralized dashboard for process monitoring and analysis
* Historical data logging for quality control and reporting
* Alert and notification system for abnormal conditions
* Integration between IoT devices and web-based management system
* Responsive dashboard for desktop and tablet usage
